# Image Dimensions Guide

To ensure your store looks professional and loads quickly, please follow these standardized image dimensions.

## 1. Store Logo
- **Dimensions**: 300px width x 100px height
- **Format**: PNG with transparent background
- **Max File Size**: 100KB

## 2. Homepage Hero Banners
- **Desktop Dimensions**: 1920px width x 800px height (or 16:9 ratio like 1920x1080)
- **Mobile Dimensions**: 800px width x 1000px height (portrait orientation)
- **Format**: JPEG (or WebP for performance)
- **Max File Size**: 300KB - 500KB

## 3. Product Images
- **Main/Gallery Images**: 1000px width x 1000px height (Square 1:1)
- **Format**: JPEG with white or very light grey background (#F8F8F8)
- **Max File Size**: 150KB - 250KB

## 4. Category / Collection Banners
- **Dimensions**: 1200px width x 400px height
- **Format**: JPEG
- **Max File Size**: 200KB

## Pro Tips for Images
- Always compress your images before uploading using tools like TinyPNG.
- Avoid placing text directly on Banner images, as it can get cut off on mobile devices. Use the CMS text overlay feature instead.
